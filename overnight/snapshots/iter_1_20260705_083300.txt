- scripts/game/upgrade_pad.gd (NUEVO): UpgradePad Area2D reutilizable.
  @export upgrade_type ("speed"|"carry"|"shelf_cap"), upgrade_name,
  base_price, max_level=5, price_growth=1.6 (precio geométrico:
  base * growth^nivel). try_buy() API pública para smoke headless:
  gasta cash via Economy.spend_cash, sube _level, aplica efecto,
  refresca labels, emite purchased. _apply_effect por tipo:
    speed → +12% move_speed base por nivel (meta base_move_speed

      para que sea idempotente y no drift).

    no maxed.
